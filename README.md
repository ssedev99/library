### 학습 내용
- ViewModel을 활용하여 UI 상태 관리 및 데이터 요청 처리
- Repository 패턴을 적용하여 UI와 데이터 영역을 분리
- Retrofit을 사용하여 Google Books API에 요청 전송
- Gson을 사용하여 API 응답 데이터 파싱
- Coil 라이브러리를 활용하여 비동기적으로 도서 이미지 다운로드 및 표시
